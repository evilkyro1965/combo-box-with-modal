Combo Box Plugin

This is the plugin to make a combo box button

Example:

Language

The snippet

HTML

<a href="javascript:;" id="language" class="combo"><span class='label'>Language</span></a>

Javascript

var data = [
	{'value':'en','label':'English'},
	{'value':'ina','label':'Indonesia'}
];

$("#language").comboModal({'data':data});
